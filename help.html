PlatypusMuertes Atlas Server Info

This is a community whitelisted server. For more information visit our discord.

How To Add Player

Edit the whitelist.js file, and add a new JSON object for the new player.

{
	"steam_id":"0000000000",
	"name":"discord name here",
	"allowed":"1",
	"notes":"your name here so I know, and any notes."
}

Be sure to add a comma when adding new JSON objects. The result should look like this: ( notice the , between },{ )

{
	"steam_id":"1234567",
	"name":"discord name here",
	"allowed":"1",
	"notes":"your name here so I know, and any notes."
},{
	"steam_id":"456789",
	"name":"discord name here",
	"allowed":"1",
	"notes":"your name here so I know, and any notes."
}

How To Remove A Players Access

Edit the whitelist.js file, and change their allowed value from 1 to 0
